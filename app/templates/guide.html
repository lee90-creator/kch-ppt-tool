설치 안내

지원 CLI 설치 및 로그인

설치 후 같은 터미널 환경에서 로그인까지 완료한 뒤 ppt-webtool 서버를 다시 시작하세요.

Claude Code

Anthropic Claude Code CLI를 전역 설치하고 로그인합니다.

설치 npm install -g @anthropic-ai/claude-code
로그인 claude 첫 실행 시 브라우저 인증을 완료합니다. 필요하면 CLI 안에서 /login을 실행합니다.

OpenAI Codex CLI

OpenAI Codex CLI를 설치하고 계정 인증을 완료합니다.

설치 npm install -g @openai/codex
로그인 codex login

Gemini CLI

Google Gemini CLI는 one-shot 실행 안정성을 위해 유료 또는 지원 대상 계정이 필요합니다.

설치 npm install -g @google/gemini-cli
로그인 gemini 시작 후 Google 로그인 방식을 선택합니다. 본 도구의 Gemini one-shot 실행은 유료 또는 지원 대상 계정이 필요합니다.

점검 순서

  1. 필요한 CLI를 설치합니다.
  2. 동일 사용자 계정에서 로그인 명령을 실행합니다.
  3. claude --version, codex --version, gemini --version 중 설치한 항목이 정상 출력되는지 확인합니다.
  4. ppt-webtool 서버를 재시작하고 생성 화면에서 사용 가능 상태를 확인합니다.
